## 삽입 정렬(Insertion Sort)

> 왼쪽 부분을 항상 정렬된 상태로 유지하며 **점진적으로 배열 전체를 정렬**

- 작은 배열이나 이미 정렬된 상태에 가까운 배열에서는 매우 빠르다.
- 온라인 알고리즘(데이터가 한 개씩 들어올 때마다 바로 정렬)에도 적합하다.

예시:

```
[5, 3, 4, 1, 2]
[3, 5, 4, 1, 2]
[3, 4, 5, 1, 2]
[1, 3, 4, 5, 2]
[1, 2, 3, 4, 5]
```

### 의사코드(Pseudocode)

1. 두 번째 요소부터 선택
2. 이전 요소와 비교하여 필요 시 교환
3. 현재 요소를 왼쪽 정렬된 부분에 올바른 위치에 삽입
4. 끝까지 반복
